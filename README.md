# Portfólio — Maria Emília Massocatto

Site estático em HTML, CSS e JavaScript para publicação no GitHub Pages.

## Publicação

1. Envie `index.html`, `favicon.svg`, `.nojekyll` e a pasta `assets` para a raiz do repositório.
2. Acesse **Settings > Pages**.
3. Em **Source**, escolha **Deploy from a branch**.
4. Selecione **main** e **/(root)**.
5. Clique em **Save**.

## Atualizações principais

- Textos e estrutura: `index.html`
- Imagens: `assets/img/`
- PDF: `assets/docs/portfolio-maria-emilia.pdf`
- Projetos do modal: objeto `projectData` no final do `index.html`

## Observação sobre o endereço

Como o repositório pertence à conta `reidosachadinhos`, o endereço padrão será:

`https://reidosachadinhos.github.io/mariaemiliamassocatto.github.io/`

Para publicar diretamente em `https://mariaemiliamassocatto.github.io/`, o repositório deve pertencer a uma conta ou organização chamada `mariaemiliamassocatto`.
